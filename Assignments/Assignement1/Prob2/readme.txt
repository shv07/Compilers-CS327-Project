

============================README=====================================

1. save the file lexer.l in the desired folder.
2. save the file on which you want to run the lexical analysis in the same folder.
3. open a terminal and change directory to the above mentioned folder.
4. type: flex lexer.l 
5. type: gcc lex.yy.c -lfl
6. type: ./a.out <name of the file to run lexical analysis on>
7. Get the output tokens.
